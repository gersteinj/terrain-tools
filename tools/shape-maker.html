Terrain Generator This is a tool to easily generate bases for tabletop gaming scatter terrain.
  • Features
  • Usage

Current features

  • Add points
  • Move points
  • Delete points
  • Move entire shape

Planned Features

  • Export SVG
  • Sizing grid
  • Sample bases for sizing
  • Order manufactured parts
  • Create multiple shapes
  • Click and drag on the edges to rearrange anchor points
  • If there isn't a point where you click, one will be created
  • To delete an anchor point, shift + click the point
  • Drag the inside of the shape to move it
  • Use the x key to export. When prompted for the filename, give a name without the extension
