Terrain Generator This is a tool to easily generate bases for tabletop gaming scatter terrain.
  • Features
  • Usage

Current features

  • Add points
  • Move points
  • Delete points
  • Move entire shape

Planned Features

  • Export SVG
  • Sizing grid
  • Sample bases for sizing
  • Order manufactured parts
  • Create multiple shapes

Click and drag to rearrange anchor points. If there isn't a point where you drag, a new point will be added. To delete an anchor point, shift + click the point. Move the entire shape by clicking inside it and dragging
